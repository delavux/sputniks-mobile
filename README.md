# Sputniks News Mobile 📰

Aplicativo mobile de notícias desenvolvido para o trabalho de **Mobile I**.


## 🚀 Como Executar

```bash
# Clone o repositório
git clone https://github.com/delavux/sputniks-mobile.git

# Entre na pasta
cd sputniks-mobile

# Instale as dependências
npm install

# Inicie o projeto
npx expo start
